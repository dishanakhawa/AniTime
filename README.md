# 🎌 AniTime - Anime Recommendation System

🚀 Live App: https://anitime-avjcvpyuvatis5jhts9psm.streamlit.app/

---

## 📌 Overview
AniTime is an AI-powered anime recommendation web application that suggests anime based on user preferences.

Built using Machine Learning and deployed using Streamlit, this project allows users to search anime by name or mood and get similar recommendations instantly.

---

## ✨ Features
- 🔍 Search anime by name or genre
- 🤖 AI-based recommendations using similarity algorithm
- 🎬 Anime posters display
- ⚡ Fast and interactive UI
- 🎨 Dark themed modern design

---

## 🧠 How it Works
- Uses **TF-IDF Vectorization** to convert text into numerical form  
- Computes similarity using **Cosine Similarity**  
- Recommends top similar anime based on user input  

---
